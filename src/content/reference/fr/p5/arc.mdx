description: >
  <p>Dessine un arc.</p>

  <p>Un arc est un morceau d'ellipse défini par les paramètres <code>x</code>,
  <code>y</code>, <code>w</code> et <code>h</code>. <br> 
  
  <code>x</code> et <code>y</code> définisent la position
  du centre de l'arc. <code>w</code> and

  <code>h</code> définissent la hauteur et la largeur de l'arc. Consultez

  <a href="/reference/p5/ellipse/">ellipse()</a> et

  <a href="/reference/p5/ellipseMode/">ellipseMode()</a> pour plus de détails.</p>

  <p>Les 5e et 6e paramètres, <code>start</code> et <code>stop</code>
  définissent les angles

  entre lesquels l'arc doit être dessiné. Les arcs sont toujours dessiné dans le sens des aiguilles d'une montre de

  <code>start</code> à <code>stop</code>. Les angles sont toujours définis en radians.</p>

  <p>Le 7e paramètre, <code>mode</code> est optionnel. Il détermine le style de remplissage de l'arc. Les styles de remplissage sont le demi-cercle (<code>OPEN</code>), le demi-cercle fermé

  (<code>CHORD</code>) ou le segment de tarte fermé (<code>PIE</code>).</p>

  <p>Le 8e paramètre, <code>detail</code> est lui aussi optionnel. Il détermine le nombre de sommets utilisés pour dessiner l'arc en mode WebGL. La valeur par défaut est 25.</p>

      background(0);

      // Style the arc.
      noStroke();
      fill(255, 255, 0);

      // Update start and stop angles.
      let biteSize = PI / 16;
      let startAngle = biteSize * sin(frameCount * 0.1) + biteSize;
      let endAngle = TWO_PI - startAngle;

      // Draw the arc.
      arc(50, 50, 80, 80, startAngle, endAngle, PIE);
